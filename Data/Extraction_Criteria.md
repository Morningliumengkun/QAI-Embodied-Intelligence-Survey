# Extraction Criteria

| RQ | Extraction Criteria |
| -- | ------------------- |
| RQ1 | The publication date of the primary study. <br> The publication venue of the primary study. <br> The venue type of the primary study. <br> The venue domain of the primary study. |
| RQ2 | The type of the quantum AI in each primary study. <br> The classical base technique of the quantum AI in each primary study. <br> The number of agents in the embodied intelligence system in each primary study. <br> The input modalities of embodied intelligence adopted in each primary study. |
| RQ3 | The integration method of quantum computing and classical AI techniques in each primary study. |
| RQ4 | The exact components of the perception–reasoning–execution workflow that are augmented with QAI. <br> The main functionality of the QAI. |
| RQ5 | The application context of the embodied intelligence in each primary study. <br> The capability and primary task of the embodied intelligence in each primary study. <br> The mobility mode of the embodied intelligence in each primary study. |
| RQ6 | The quantum-side execution of the quantum AI algorithm in each primary study. <br> The quantum platform used for the quantum-side execution in each primary study. <br> The embodied-side platform used for the embodied-side execution in each primary study. |
| RQ7 | The availability of open-source repositories and implementation platforms used in the primary studies. |
